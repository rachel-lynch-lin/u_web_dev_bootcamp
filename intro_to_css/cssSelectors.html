CSS Selectors

The Basics: Element, ID, and Class

Element Selector

  • Select all instances of a given element
  • Nothing goes before the name of the element.

ID Selector

  • Selects an element with a given ID. Only one per page! But there can be multiple ID's per page.
  • A # precedes the name of the ID.

Class Selector

  • Selects all elements with a given class.
  • A . precedes the name of the Class.
