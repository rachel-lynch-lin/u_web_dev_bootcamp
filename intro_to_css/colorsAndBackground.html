Colors and Background in CSS

  • Use color to set text color and background for background color.
  • The div element has a background image which is added with a url.
  • The size of the background image made to fit the size of the page.
  • This can be done instead of having a smaller image repeat many times across a page.
>

Borders

  • Borders in CSS have three different parts:
    1. Width
    2. Color
    3. Style
  • Borders will not work unless all three are set.
  • These can be listed on their own line or on a single line.
